Travelers companion is a front-end app powered by Expedia API that allows users to search for the cheapest flights.

<b>Getting Started</b>

	Installing

    Git clone https://github.com/nfcortega89/CS1
    Cd CS1
    Npm install

  Launching
    <li>Npm start</li>

  Testing
    <li>Npm test</li>
